##分析设计
为了减少多版本更新时菜单冲突覆盖的问题，管理后台的菜单改成从数据库读

##系统设计

###代码设计
Manage.php中initMenu方法改写，对从数据库中读取获得menu数据进行排序及返回。
增加data类Menu.php对数据库menu表的操作

###测试重点
查看各个菜单节点是否正确，是否有重复菜单，新增菜单是否在指定的一级菜单目录下

###SQL相关
db_p2p库增加tb_menu表，每次新增菜单需要通过insert into语句执行。
