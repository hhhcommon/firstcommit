## 分析设计  
在后台提供退款的工具  

## 详细设计  
1. 客服输入订单号、用户ID、退款金额、退款原因即可发起退款请求  
2. 查看退款状态  

## 部署步骤  

### 1. 执行SQL  
依次执行  **/README/SQL/** 目录下的sql语句  
`153.lyq.sql`  
`157.lyq.sql`  

### 2. 添加配置  
在 **/conf/globals.php** 中添加配置  
> `$GLOBALS['CONF']['dbByObj']` 数组中添加一项： `'chargeback' => [2, 'default'],`  

### 3. 覆盖文件  
将工具包中的文件覆盖原文件即可
